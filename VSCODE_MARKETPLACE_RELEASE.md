# Unfold v0.2.0

**Hierarchical plans as markdown files — AI-generated, git-native, human-readable**

## Features

- **AI-powered plan generation** — Type `@unfold /create` and describe what you want to build
- **Visual tree view** — See your entire plan hierarchy in the sidebar with status indicators
- **Rich webview** — Click any node to read details with syntax highlighting and Mermaid diagrams
- **Git-native** — Plans are plain markdown files, version-controlled alongside your code
- **Workspace-aware** — The AI analyzes your project structure before generating plans

## What's Changed

### v0.2.0
- Improved plan generation with delimiter-based output format
- Better prompt engineering for more reliable LLM responses
- Streamlined codebase and type definitions
- Updated marketplace categories for better discoverability

## Commands

| Command | Description |
|---------|-------------|
| `@unfold /create` | Generate a new plan from a description |
| `@unfold /convert` | Convert an existing plan to Unfold format |
| `@unfold /refine` | Expand or restructure an existing plan |
